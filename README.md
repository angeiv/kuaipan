Introduction
------------

Python API and fuse file system for kuaipan.cn

- kuaipan.py - python API for kuaipan.cn
- main.py - Fuse file system for kuaipan.cn


Requirements
------------

- requests
- requests\_oauthlib
- json


Install
-------

    pip install -r requirements.txt

Usage
-----

Mount yout kuaipan to given mount directory:

    main.py <mount point>
